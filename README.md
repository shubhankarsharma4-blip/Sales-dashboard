# Sales Growth & Expansion Dashboard

Interactive Streamlit dashboard for sales performance, margin analysis, concentration risk, and expansion planning.

## Run locally

```powershell
py -m pip install -r requirements.txt
py -m streamlit run main.py
```

## Deploy to Streamlit Community Cloud

1. Push this project to a GitHub repository.
2. Sign in to Streamlit Community Cloud.
3. Create a new app and select your GitHub repo, branch, and `main.py` as the entrypoint.
4. Streamlit will install packages from `requirements.txt` automatically.
5. After deployment, use the app settings page to redeploy when you push updates.
